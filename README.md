# 餐廳清單
此網站是由Exepress + Node.js + sequelize + mySQL 製成的餐廳清單網站，可以瀏覽清單頁面、新增、刪除、編輯餐廳清單內容。

## Features - 產品功能

1. 使用者可以使用搜尋功能查詢餐廳。
2. 使用者可以新增一間餐廳頁面資訊。
3. 使用者可以編輯現有的餐廳頁面資訊。
4. 使用者可以刪除現有的餐廳頁面資訊。
